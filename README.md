#xGallerify

A lightweight, responsive, smart gallery based on jQuery.

[Demo](https://rawgit.com/xremix/xGallerify/master/Sample.html)

##Usage:

```JS
		$('.photos').gallerify({
			 margin:5,
			 mode:'default',
			 // mode:'bootstrap',
			 // mode:'flickr',
			 // lastRow:'fullwidth',
			 lastRow:'adjust',
		});	
```
